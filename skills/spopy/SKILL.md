name: spopy
description: Control Spotify from the terminal via spopy ‚Äî playback, search, queue, playlists, devices, and library.
---

Use `spopy` directly.

Common commands:
- `spopy play "song name"`
- `spopy pause`
- `spopy resume`
- `spopy next`
- `spopy previous`
- `spopy status`
- `spopy search "query"`
- `spopy queue add "song"`

Accepts Spotify URI, URL, ID, or search query.
Auto-selects a device when only one is available.
Run `spopy --help` for everything else.
